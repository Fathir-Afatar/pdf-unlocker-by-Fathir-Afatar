# How to Use PDF Unlocker by Fathir



## Step-by-step:

1. **Splash screen** akan tampil selama 2 detik saat aplikasi dijalankan.
![Splash Screen](images/Splash%20Screen.png)
---

2. Klink tombol **"Pilih PDF"** untuk memilih file PDF yang terkunci.
![Pilih PDF](images/Pilih%20Folder%20PDF.png)

---

3. Pilih file **password.txt**, file notepad yang berisi daftar password.
![Password](images/pilih%20notepad%20password.png)

---

4. Klik tombil **"Mulai Proses"**.
![Proses](images/Processing.png)

---

5. Jika berhasil, password akan ditampilkan dan file PDF akan disimpan ulang tanpa password.
![Hasil](images/Proses%20Selesai.png)

---

## Extra Features:
- Klik label **"MIT LICENSE (c) Fathir"** di Bawah untuk  melihat lisensi penggunaan.
![MIT License](images/License.png)

- Jika file 'License' tidak ditemukan, aplikasi akan menampilkan pesan kesalahan.

## Tips:
- Gunakan susunan password dalam file .txt yang sesuai dengan susunan file PDF.
- Hindari mengganti nama file output menjadi sama dengan input PDF, untuk menghindari penimpaan.
